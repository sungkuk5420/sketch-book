# sketch-book
Vanilla JS만 사용하여 스케치북을 만들어 보았습니다. 

Painting Board made with Vanilla JS

Vanilla JSだけ使ってスケッチブックを作ってみました。

## Demo / 데모 / デモサイト

[Link Open](https://sketch-book-web.netlify.app/)

* 배경색 변경
* 모바일 대응
