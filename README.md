# sketch-book
Vanilla JS만 사용하여 스케치북을 만들어 보았습니다. 

Painting Board made with Vanilla JS

## Demo / 데모

[라이브 데모 보기](https://sketch-book-web.netlify.app/)

## 기능

* 그림그리기
* 이미지저장
* 배경색 변경
* 모바일 대응
